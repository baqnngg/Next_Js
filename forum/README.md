# Nextjs/forum

# Introduction
Next.js를 활용하여 커뮤니티 게시판 제작을 실습하였습니다.
MongoDB Atlas를 사용하여 CRUD기능을 추가하였습니다.

# Getting Started
#Introduction
1. Repository Clone
```bash
git clone === 
```

2. install npm packages
```bash
npm install
```

3. Start Development Server
```bash
npm run dev
```

4. Open Browser
```
http://localhost:3000
```
